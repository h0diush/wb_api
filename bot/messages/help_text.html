Привет! Я бот, который поможет вам следить за изменением цен продуктов на Wildberries и сохранять товары для дальнейшего просмотра и отслеживания. Для того, чтобы начать использовать бота, вы можете использовать следующие команды: /new_product - добавить новый продукт для отслеживания цены /my_products - просмотреть список сохраненных продуктов /delete_product - удалить продукт из списка сохраненных продуктов /reviews - просмотреть отзывы о выбранном продукте Также вы можете использовать команду /help, чтобы просмотреть это сообщение еще раз. Если у вас возникнут какие-то вопросы или проблемы с ботом, вы можете написать нам в службу поддержки.
